COMANDOS DE BIBLIOTECAS DE REACT:

Biblioteca para convertir color hexadecimal a RGBA:
npm i hex-to-rgba

Biblioteca para generar ID:
npm i uuid

Biblioteca de React de iconos:
npm install react-icons --save

Instalar Styled Componets:
npm install styled-components

Libreria de Material UI:
npm install @mui/material @emotion/react @emotion/styled

COMANDOS DE JSON SERVER:

Instalar Json Server:
npm i -g json-server

Crear una ruta para el db.json:
json-server --watch .\db.json --port=5000
